# Payroll Management System (Java)

## Detailed Description

The **Payroll Management System** is a Java console application designed to help manage employee information and calculate payroll efficiently.
The system allows users to add employees, view employee records, generate salary slips, and delete employee data. It calculates the **net salary based on the employee's basic salary**, demonstrating how payroll systems work in real-world organizations.
This project applies important Java programming concepts including **Object-Oriented Programming (OOP), collections, and modular program structure**.

The application is divided into multiple components such as:

* Employee class (stores employee details)
* PayrollService class (manages employee records)
* PayrollCalculator class (performs salary calculations)
* Main class (user interaction and system menu)

## Features

*  Add new employees
*  Store employee details (ID, Name, Designation, Salary)
*  View all employee records
*  Generate salary slip for employees
*  Delete employee records
*  Interactive menu-driven system

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* ArrayList / List Collections
* Console-based user interface
* Modular class design

## How to Run

 Install Java (JDK 8 or higher)
 
 Compile the program:
javac Main.java

Run the program:

java Main
